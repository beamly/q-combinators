# node-q-combinators

functions to combine (q) promises. 


## Installing

Add the following to your package.json dependencies: 

```json
{
	"dependencies": { 
		"q-combinators": "git+ssh://git@github.com:zeebox/node-q-combinators.git#v0.0.1"
	}
}	
```
